# Инструкция по добавлению фотографий

Положите в эту папку 4 фотографии с такими названиями:

- **misha.jpg** - фото для Миши
- **sergo.jpg** - фото для Серго
- **dmitry.jpg** - фото для Дмитрия
- **danya.jpg** - фото для Дани

## Поддерживаемые форматы:
- .jpg / .jpeg
- .png
- .webp

## Рекомендации:
- Размер: оптимально 800x1000px или больше
- Соотношение сторон: примерно 4:5 (вертикальное фото)
- Качество: не сжимайте слишком сильно, минимум 80%

После добавления фотографий они автоматически появятся на сайте!
