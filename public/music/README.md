# Музыка для фона

Поместите файл с фоновой музыкой в эту папку с названием:
- `background.mp3`

Музыка будет автоматически воспроизводиться при загрузке сайта.

**Примечание:** Большинство современных браузеров блокируют автовоспроизведение звука до первого взаимодействия пользователя с сайтом. Если автовоспроизведение заблокировано, музыка начнет играть после первого клика на странице.
